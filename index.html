Configura tu API Key

Para analizar las imágenes esta app usa la inteligencia artificial de Anthropic (Claude). Necesitas una API key gratuita.

Cómo obtener tu API key

1

Entra a console.anthropic.com y crea una cuenta gratuita

2

Ve a API Keys → Create Key

3

Copia la clave (empieza con sk-ant-) y pégala aquí abajo

4

La clave se guarda solo en este dispositivo y no se comparte con nadie

📲

Instalar como app

En Chrome: menú (⋮) → "Añadir a pantalla de inicio"

📷

Fotografía de la ruma

Toca aquí para tomar foto o elegir de la galería.
Debe tener una línea blanca vertical de 1 metro como escala.

🔬 ¿Cómo tomar la foto? Coloca un jalón o vara de 1 metro al costado izquierdo de la ruma. Captura el extremo de los troncos de frente con el jalón visible.
